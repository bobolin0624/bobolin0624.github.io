JS30 系列筆記 - Day 11 ~ Day 15

Jun 27, 2022 in Note

走了一半的 JS30，遇到不少沒用過的方法，像是 Day 11 的影片播放，覺得很有趣。

這也算是上這系列的目的，多讓自己對沒使用過的方法有些印象，未來遇到時就不會完全陌生了～

Table of Contents

  1. D11 - Custom HTML5 Video Player
  2. D12 - Key Sequence Detection (KONAMI CODE)
  3. D13 - Slide In on Scroll
  4. D14 - Object and Arrays - Reference VS Copy
  5. D15 - LocalStorage and Event Delegation

D11 - Custom HTML5 Video Player

目標：使用 JavaScript 製作網頁播放器的按鈕

播放與暫停：video.paused() / video.play()

快轉或倒退：使用 dataset 去抓取使用者的行為，控制 currentTime 的計算

音量與速度：使用 input value 控制 slide bar

時間條跟著播放時間走：利用 currentTime 佔全影片的比例在 CSS 使用 flexBasis 計算現在時間為影片時間的百分之幾

時間條控制狂：監聽 mousemove 點擊 progress bar 要到指定的時間 (scrub) 影片秒數，利用 offsetX 佔時間條的寬度比例來計算，改變 video.currentTime

My practice demo

D12 - Key Sequence Detection (KONAMI CODE)

目標：判斷 user 輸入的 key 達到特定字串後，output 做出相對反應（猜字遊戲之類的？）

取字串跟 PJ 大大一樣使用 Array.slice() 來做

另使用 Array.prototype.join() 將陣列合併為字串

My practice demo

D13 - Slide In on Scroll

目標：頁面滑動到特定高度時，圖片會自動滑入畫面中，使用 opacity (0 → 1)

如何計算圖片何時要準備滑入（利用高度）：window.scrollY 給的是視窗上方到捲軸最上面的距離 + 視窗高度 = 最底部的（圖片準備要出來的高度）

使用 offset 屬性前，盡量先用 offsetParent 來查看該元素的父層對應到的是哪個元素

作者內建了一個 debounce 函式，作用是在設定好的時間內，只會觸發一次事件，暫時還沒有深入研究。

D14 - Object and Arrays - Reference VS Copy

目標：了解 JS 的重要觀念

關於 變數 / 物件 / 陣列 的 call by reference 及 call by value

可以參考這篇 Huli 大大寫的極致探討文章

D15 - LocalStorage and Event Delegation

目標：練習將資料暫存於 localStorage 中，讓頁面刷新時可以讀取

preventDefault() ：避免表單送出時，跳轉網頁重新整理

監聽 parent 對 checkeboxes 做 done or !done
